# Process

## Definitions
**Definition of Ready**
- One-sentence outcome
- Acceptance criteria listed
- Test idea stated
- Size set; no unowned deps

**Definition of Done**
- Unit + relevant integration tests pass
- CLI example works on fixtures
- Docs updated
- Clear logs/errors; follow-ups ticketed

## Daily Flow
- Set Daily Goal → break into 2–4 micro-tasks
- TDD loops (Red-Green-Refactor) + docs touch
- Verification: run fixture E2E
- PR checklist in repo
